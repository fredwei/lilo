<p>此插件为依赖jquery的一个全屏左进左出的特效</p>

<p>不适用于所有情况，是为一个网站特别做的</p>

<p>其中或有些技术不到位，做得不好的，欢迎指教!</p>

<p>轮播区域为li结构，其中有三块内容</p>

<ul>
	<li>一块背景，命名为slide-bg</li>
	<li>一块小图，命名为slide-img</li>
	<li>一块文字，，命名为slide-txt</li>
</ul>

<p>可单独对着三块内容做控制当然也可以修改源码，欢迎折腾。</p>
